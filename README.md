# Digital Forensic Static Writeup

Static HTML/CSS site for the "AoTR 4: A Drone in the Snow" writeup.

## Files

- `index.html` - main page
- `styles.css` - site styling
- `1.png` to `9.png` - local screenshots used in the walkthrough

## Deployment

This repository includes a GitHub Actions workflow for GitHub Pages deployment.

After pushing:

1. Open the repository settings on GitHub.
2. Go to **Pages**.
3. Set the build source to **GitHub Actions** if it is not already selected.
4. Wait for the workflow to finish.

The site will then be available at:

`https://dipszy.github.io/Digital-Forensic/`
